### 2. 데이터 전처리 : csv -> json 변환
```
import pandas as pd
import ast
import json

# Load the data
df = pd.read_csv('movies_metadata.csv', low_memory=False)

# Clean data (drop rows with missing values, incorrect types, etc.)
df = df.dropna(subset=['vote_count'])
df['vote_count'] = df['vote_count'].astype(int)

# Process the 'genres' column
def extract_genres(genres_str):
    # Using ast.literal_eval to safely parse the string as a Python literal
    genres_list = ast.literal_eval(genres_str)
    return [g['name'] for g in genres_list]

df['genres'] = df['genres'].apply(extract_genres)

# Sort by 'vote_count' and take the top 100
df = df.sort_values(by='vote_count', ascending=False)
df_top100 = df.head(100)

# Convert the DataFrame to JSON string and pretty-print it
json_str = df_top100.to_json(orient='records')
json_data = json.loads(json_str)  # convert JSON string to Python object

with open('top100_movies.json', 'w') as file:
    json.dump(json_data, file, indent=4)
```
- vote_count 순으로 100개의 영화만 사용
- 장르 데이터 정리
- json tree형태로 정리
